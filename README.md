## Дипломный проект Movie Explorer бэкенд

## Описание
------------------
Backend приложение со следующими возможностями: 
* авторизация и регистрация пользователей 
* поиск фильмов по стороннему API 
* обновление пользовательских данных. 
-------------------------------------------- 
Целью данной практической работы является демонстрация использования  следующих технологий:   
* API (POST, GET, PATCH, DELETE)
* Node.js
* Express 
* Mongoose

Адрес репозитория: https://github.com/Elizaveta-Parada/movies-explorer-api/tree/level-1
